# Projeto de Testes Manuais - SauceDemo

Este projeto foi desenvolvido com o objetivo de praticar testes manuais em uma aplicação web.

## Cenários testados:

- Login sem credenciais
- Login com senha vazia
- Login com credenciais inválidas
- Adição de produto no carrinho
- Remoção de produto

## Tipos de teste:

- Testes funcionais
- Validação de campos
- Fluxo de compra

## Objetivo:

Desenvolver habilidades em QA (Quality Assurance), incluindo:
- Criação de casos de teste
- Validação de funcionalidades
- Identificação de cenários de erro
